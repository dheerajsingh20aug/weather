# 5 day forecast

"Build a single HTML page displaying the 5 day weather forecast"

## Technology stack

* ES6
* React
* Babel
* Node
* [OpenWeatherMap 5 day forecast](http://openweathermap.org/forecast5)

## Building
Assuming node 6.5.0

```
npm install
npm run build
npm start
```
This will start up the app.
Navigate to http://localhost:9002
